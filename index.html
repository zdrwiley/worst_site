CSS Syntax

The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

How to Use CSS Flexible Box Layout or Flexbox or The Internet is Boxes

You should put everything in your site into a box. This is important because everyone uses the internet on phones now.
Hover your cursor here.
Huzzah! You can make random text appear on hover for no reason if you'd like.
How to: just create lots of "divs" in your HTML file. Be sure to name them all "div," and don't add any classes or make them unique in any way or explain in any way what any of the divs are for. Then go into your CSS and add lots of :required tags for styling.
Also don't worry about resizing any images you want to use or trying to make them display properly across different screen sizes and resolutions. Just throw those sweet pics right in there and don't give it a second thought.
